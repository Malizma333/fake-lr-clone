### Car Mode

This is a separate branch for car mode, which forms a new sprite from the bosh texture and adds a special trail.

| Action       | Key         |
|--------------|-------------|
| Speed Up     | W           |
| Slow Down    | S           |
| Turn Left    | A           |
| Turn Right   | D           |
| Drift Left   | Left Arrow  |
| Drift Right  | Right Arrow |
| Zoom In      | Up Arrow    |
| Zoom Out     | Down Arrow  |
